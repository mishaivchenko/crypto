📡 *Сигналы к фандингу*

━━━━━━━━━━━━━━━━━━━━
*SignalCandidate* — входящий сигнал из API фандинга\.

🔁 *Источник:* `uainvest.com.ua/api/funding`
Опрашивается каждые 60 секунд\. Создаёт записи `SignalCandidate`\.

━━━━━━━━━━━━━━━━━━━━
📊 *Статусы сигнала:*

🕐 `PENDING` — только что создан, ожидает нормализации
✅ `NORMALIZED` — символ успешно распознан
❌ `FAILED` — ошибка нормализации символа
📋 `EVENT_CREATED` — одобрен, создано событие
🚫 `REJECTED` — отклонён оператором
🗑 `DELETED` — удалён

━━━━━━━━━━━━━━━━━━━━
🔍 *Дедупликация:*
Система проверяет `sourceMessageId`\. Дубликаты обновляются, а не создаются повторно\.

⚡ *Нормализация:*
Маппинг биржевых символов в канонический формат через `InstrumentRegistryService`\.
