🏦 *Биржи \(Venues\)*

━━━━━━━━━━━━━━━━━━━━
Поддерживаемые биржи: *Bybit, Gate\.io, OKX, KuCoin, Bitget*

━━━━━━━━━━━━━━━━━━━━
🔧 *Режимы доступа:*

🧪 `testnet` — тестовая сеть биржи
🔴 `production` — боевой режим

━━━━━━━━━━━━━━━━━━━━
🔑 *Учётные данные:*
Хранятся в зашифрованном виде \(AES\-GCM\)\. Никогда не возвращаются через API в открытом виде\.

Параметры:
• `api-key` — API ключ
• `secret-key` — секретный ключ
• `passphrase` — только для OKX, KuCoin, Bitget

━━━━━━━━━━━━━━━━━━━━
📊 *Диагностика биржи:*
Каждая биржа имеет:
• Статус соединения \(`OK` / `ERROR`\)
• Среднее время запроса
• Количество активных инструментов
• Время последней синхронизации

━━━━━━━━━━━━━━━━━━━━
⚠️ *Geo\-блокировки:*
Bybit заблокирован для UA IP\-адресов\. Используй Gate\.io или другую биржу как основную\.
