📐 *Архитектура системы*

━━━━━━━━━━━━━━━━━━━━
Система состоит из трёх модулей:

🖥 *Monitor App* `:8090`
Операторская панель управления\. Принимает сигналы, управляет событиями и трейдами\. Хранит всё в SQLite\. Содержит Web UI\.

⚙️ *Engine App* `:8091`
Лёгкий исполнительный движок\. Читает план из монитора и отправляет ордера на биржу\. Без собственной БД\.

📦 *Platform Core*
Общая библиотека: иммутабельные доменные объекты, контракты, утилиты\.

━━━━━━━━━━━━━━━━━━━━
🔄 *Поток данных:*

`Funding API → Сигнал → Оператор → Событие → Трейд → Engine → Ордер`

1️⃣ API опрашивается каждые 60 сек
2️⃣ Оператор проверяет и подтверждает сигнал
3️⃣ Создаётся FundingEvent
4️⃣ Оператор вооружает трейд
5️⃣ Engine исполняет ордер

━━━━━━━━━━━━━━━━━━━━
🛡 *Безопасность по умолчанию:*
Все режимы выключены\. Live\-ордера невозможны без явного включения\.
