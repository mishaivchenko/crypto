⚙️ *Engine App*

━━━━━━━━━━━━━━━━━━━━
Лёгкий исполнительный движок — отдельный Spring Boot сервис на порту `:8091`\.

━━━━━━━━━━━━━━━━━━━━
🔄 *Как работает:*

1\. `EngineExecutionScheduler` тикает каждые 250 мс
2\. Читает актуальный план из монитора \(`/internal/engine/plans`\)
3\. `EngineExecutionService` проверяет время входа
4\. При наступлении времени — отправляет ордер через биржевой адаптер
5\. Записывает `OrderAttempt` в монитор
6\. Публикует метрики через `EngineMetricsPublisher`

━━━━━━━━━━━━━━━━━━━━
🛡 *Защита от случайного исполнения:*

По умолчанию ВСЁ выключено:
• `ENGINE_EXECUTION_LOOP_ENABLED=false` — цикл исполнения выключен
• `ENGINE_LIVE_ORDER_ENABLED=false` — live\-ордера выключены

━━━━━━━━━━━━━━━━━━━━
🔒 *Изоляция:*
Engine не может изменить состояние `FundingEvent` или `ArmedTrade` напрямую\. Только читает план и записывает результаты\.

━━━━━━━━━━━━━━━━━━━━

Каждый production\-класс Engine покрыт mutation\-тестами на 100%\. 13 core\-классов, каждый задокументирован в `docs/engine-tdd/`\.
