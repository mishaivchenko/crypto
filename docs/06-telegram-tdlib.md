# Telegram via TDLib

## Что делает
- читает сообщения из канала
- извлекает символы (например BTC/USDT)
- кладёт в watchlist

## Ограничения
- TDLib = нативная библиотека
- текущая реализация работает на macOS
- цель: собрать TDLib под Linux и положить в Docker image

## Хранилище сессии
- /data/tdlib (volume)
- в контейнере должен сохраняться state (иначе каждую перезагрузку login)
